## What is this?

A unified package of AI agents and skills for the **OSDU** (Open Subsurface Data Universe) platform ecosystem. One `apm install` gives your AI coding assistant deep knowledge of OSDU infrastructure, platform services, and development workflows.

<CardGrid stagger>
  <Card title="8 Specialist Agents" icon="puzzle">
    Platform orchestrator, infrastructure specialists (CIMPL and SPI), QA sub-agents for test execution, failure analysis, environment comparison, and reporting.
  </Card>
  <Card title="28 Skills" icon="open-book">
    From cloning repos to scanning dependencies, running QA tests, reviewing merge requests, managing knowledge vaults, and shipping changes — the full developer lifecycle.
  </Card>
  <Card title="Every AI Tool" icon="laptop">
    Deploys to GitHub Copilot, Claude Code, Cursor, OpenCode, and Codex from a single source via APM (Agent Package Manager).

    Six-layer test framework validates structure, trigger accuracy, multi-turn workflows, and skill value — across both Copilot and Claude.

## Who is it for?

**Platform Engineers** working with CIMPL or SPI infrastructure — Terraform, Helm, AKS, Azure PaaS services, environment provisioning, and deployment debugging.

**Service Developers** contributing to OSDU platform services — building, testing, scanning dependencies, reviewing MRs, and shipping changes across 30+ GitLab repositories.

## Quick Start

```bash
# Install APM
curl -sSL https://aka.ms/apm-unix | sh

# Add to any project
apm install danielscholl-osdu/osdu-spi-skills

# Done — your AI tool is now OSDU-aware
```
